# OpenZeppelin
Ethereum ERC20 token with truffle OpenZeppelin

## Run blockchain
- testrpc
- [import account in MetaMask](http://truffleframework.com/tutorials/pet-shop#interacting-with-the-dapp-in-a-browser) 

## Installation
- npm install
- truffle compile
- truffle migrate



## Play with contract
- truffle console

## Start WebApp
- npm run dev
